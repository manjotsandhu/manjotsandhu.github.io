Personal Projects

Whado - Android App

Android application that allows users to create and respond to custom real time polls about a variety of topics. Polls are searchable by category, users can follow users and view polls on their home feed. It utilizes FireBase on the backend to store and process poll data and user data. The App also makes use of FireBase’s Social Media API’s and enables WhaDo to integrate with Google+ and Facebook for user account creation and user login. It was built over 4 month time period in collaboration with two classmates and is currently still being refined and developed.

Aqua Monitor - Web App

Web application that tracks and displays real time aquarium data from a aquarium controller with an existing API. Users can interact with historical data from various components of the controller and view real time updating information on their aquarium. It also contains a fully functional user management system with options for periodic updates with select aquarium data specified by the user. Aqua Monitor was built using Javascript, HTML, and CSS with the use of FireBase, Material Boostrap, and HighCharts.js.

CDatum - Web App

Web application that was developed for a Hackathon in collaboration with 4 team members. CDatum is a platform that uses a blockchain transaction ledger to track donations to charities. Our aim was to attempt to address the issue of transparency within charity organizations. We built this application using IBM's hyperledger framework for the blockchain transactions, and Angular.js on the frontend.
